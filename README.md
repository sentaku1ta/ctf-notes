## ユーザー名・アカウント調査
- Namechk
  - 複数SNS・サービスでユーザー名が使われているかを確認できる
  - 初動調査で有効
  - https://namechk.com/
- WhatsMyName
  - OSINT向けに作られたユーザ名横断検索ツール
  - Web UIで検索可能
  - https://whatsmyname.app/
